What's Inside

Platform Features

CPU Scheduling Engine

Full C-compiled WebAssembly engine running FCFS, SJF (preemptive & non-preemptive), Round Robin, Priority, and AI-driven scheduling — all inside the browser.

WASM · C · Browser-native

Neon Gantt Chart

Real-time animated Gantt visualization. Each process gets a unique neon color lane with time-travel scrubbing — replay any simulation tick by tick.

Real-time · Interactive

AI Scheduler + RAG Chat

An AI agent picks the optimal algorithm for your workload. Ask questions in the terminal chat powered by Retrieval-Augmented Generation — get instant algorithm explanations.

RAG · LLM-powered

Gamification HUD

Earn XP for every simulation run. Collect achievement badges. Challenge the AI in Manual Duel mode — pick your own scheduling order and see if you beat the machine.

XP · Badges · Duel Mode

Time-Travel Playback

Scrub through any simulation frame-by-frame. Watch processes transition between Ready → Running → Waiting → Done states with full rewind control at 1×, 2×, 5× speed.

Timeline · Scrubber · Replay

Performance Gauges

Live CPU utilization, average wait time, turnaround time, and throughput gauges. Compare algorithm efficiency visually with animated ring meters.

Metrics · Analysis · Compare

Virtual Desktop + Algorithm Book

A fully interactive virtual OS desktop experience with draggable windows. Open the Algorithm Book to browse every scheduling algorithm with complexity analysis, pseudocode, and examples. The Algorithm Sandbox lets you step through each algorithm manually — the closest you can get to running an OS kernel in the browser.

Virtual OS · Draggable UI · Deep Dives
